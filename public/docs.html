Architecture Overview

A high-performance, serverless data pipeline designed to ingest, process, and serve massive scale Aadhaar ecosystem metrics without database overhead.

Core Components

Smart Ingestion

Automated Python jobs running on GitHub Actions fetch partial updates from Data.gov.in using intelligent exponential backoff and parallel threading.

Release-as-Database

Instead of a traditional DB, processed CSV bundles (Raw & Master) are versioned and stored in GitHub Releases, acting as an immutable, high-bandwidth CDN.

Zero-Latency API

FastAPI acts as a lightweight proxy, redirecting PowerBI streams directly to the pre-compressed assets in GitHub, eliminating server timeouts.

API Reference

GET /api/integration/powerbi

Primary Integration Stream. Redirects to the latest `master_dataset_final.csv`. Used by the PowerBI Report to pull 500MB+ data in seconds.
