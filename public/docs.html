UIDAI Aadhaar Insights API Docs

Base URL: https://uidai.sreecharandesu.in/api

Overview

Advanced API for fetching real-time Aadhaar insights from official source (data.gov.in) with high-speed Redis caching. Secured with API keys and optimized for serverless performance.

Available Datasets & Fields
enrolment

New Aadhaar enrolments by age group.

fields: ["date", "state", "district", "pincode", "age_0_5", "age_5_17", "age_18_greater"]
demographic

Demographic updates (address, DOB, etc).

fields: ["date", "state", "district", "pincode", "demo_age_5_17", "demo_age_17_"]
biometric

Biometric updates (fingerprint, face, iris).

fields: ["date", "state", "district", "pincode", "bio_age_5_17", "bio_age_17_"]
Insights Query Endpoint
POST
/api/insights/query
Search and filter insights
Authentication
HeaderTypeRequiredDescription
X-API-KEYstringYesYour confidential Client API key
Query Parameters (JSON Body)
FieldTypeDescription
datasetstringOne of: enrolment, demographic, biometric
select NEWarrayArray of fields to return (e.g. ["state", "age_0_5"])
filtersobjectKey-value pairs for hard filtering (e.g. {"State": "Telangana"})
limitnumberRecords per page (default: 100)
pagenumberPage offset (default: 1)
Example Payload
{
  "dataset": "enrolment",
  "select": ["state", "district", "age_0_5"],
  "filters": {
    "state": "Telangana"
  },
  "limit": 5
}

Implementation Examples
curl -X POST https://uidai.sreecharandesu.in/api/insights/query \
-H "Content-Type: application/json" \
-H "X-API-KEY: YOUR_KEY" \
-d '{"dataset": "enrolment", "limit": 1}'
curl -X POST https://uidai.sreecharandesu.in/api/insights/query \
-H "Content-Type: application/json" \
-H "X-API-KEY: YOUR_KEY" \
-d '{"dataset": "biometric", "select": ["state", "bio_age_17_"], "limit": 1}'
